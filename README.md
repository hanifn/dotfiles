# dotfiles
My dotfiles configuration

## Dependencies
- [Homebrew](https://brew.sh/)
- Xcode
- Select Xcode app CLI tools `sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer`

## Install

```
git clone https://github.com/hanifn/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```
